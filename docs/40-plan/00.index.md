# 40-plan 인덱스

이 디렉토리는 AI 처리 과정에서 만들어지는 plan의 구조와 실제 코드 산출 방식을 정의한다.

## 파일 목록

- `01-plan-structure.md`: PM, CTO, PL, PA가 공유하는 plan 구조
- `02-communication.md`: 역할 간 소통과 전달물 기준
- `03-code-output.md`: 실제 코드 산출 구조와 변경 단위 기준

## 책임

- 작업 계획의 공통 포맷을 정의한다.
- 역할 간 전달물이 누락되지 않도록 한다.
- 코드 변경 전 산출 구조를 먼저 합의한다.
- 실행 결과가 리뷰 가능한 단위로 나오도록 한다.
