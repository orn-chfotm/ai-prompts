# 30-execution 인덱스

이 디렉토리는 `20-ai-process`의 역할 검토와 사용자 승인이 끝난 뒤, 실제 작업을 어떻게 진행할지 정의한다.

## 파일 목록

- `01-workflow.md`: 승인 이후 실행 흐름과 되돌림 기준
- `02-task-splitting.md`: PA 작업 분리와 병렬 실행 기준
- `03-output-format.md`: 실행 중 보고와 산출물 형식

## 책임

- 승인된 작업을 실행 가능한 절차로 바꾼다.
- PA 작업 단위를 정의한다.
- 실행 중 중단, 보고, 에스컬레이션 기준을 둔다.
- 실행 결과를 PL, CTO, 사용자에게 전달 가능한 형태로 정리한다.
